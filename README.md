## Project Title

Student API CRUD using Laravel (api-call)

## Short Project Description

This project is a REST API CRUD application developed using Laravel. It allows users to create, retrieve, update, and delete student records using API requests tested in Postman.

## Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/leoloves2004/api-call.git
```

### 2. Open the Project

```bash
cd student_api
```

### 3. Install Dependencies

```bash
composer install
```

### 4. Create Environment File

```bash
copy .env.example .env
```

### 5. Generate Application Key

```bash
php artisan key:generate
```

### 6. Run Database Migration

```bash
php artisan migrate
```

### 7. Start Laravel Server

```bash
php artisan serve
```

### 8. Test API in Postman

Base URL:

```text
http://127.0.0.1:8000/api/students
```
